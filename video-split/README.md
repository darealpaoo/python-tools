# Yêu cầu

Python 3.11 trở lên.

## Cài đặt

```bash
pip install moviepy
pip install psutil
```
Hoặc có thể chạy file install.bat để cài đặt các module cho nhanh.

## Cách sử dụng

```
1. Cho video muốn tách vào trong folder tool.
2. Chạy file video_split.py và làm theo yêu cầu.
3. Demo video: Đang cập nhật...
```
